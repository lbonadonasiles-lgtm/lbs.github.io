Energy Demand Forecasting

Completed

Built a machine learning pipeline to predict household electricity consumption using the UCI Electric Power Consumption dataset (~2M rows). Explored seasonal and daily usage patterns through visualization, engineered time-based and lag features, then trained and compared three models — Linear Regression, Random Forest, and XGBoost. XGBoost achieved the best performance with an R² of 0.61 after feature engineering, capturing seasonal trends and daily usage cycles.

Heart Disease Risk Prediction

Completed

Built a classification pipeline to predict heart disease presence using clinical patient data from 918 records. Analyzed relationships between chest pain types, ECG readings, and cardiac risk. Trained Logistic Regression, Random Forest, and XGBoost — Logistic Regression achieved the best accuracy at 89.7% with a disease recall of 92.2%, correctly identifying 94 out of 102 heart disease patients.

EEG Seizure Detection

Completed

Built a machine learning pipeline to detect epileptic seizures from EEG brain recordings (11,500 samples, 178 data points each). Engineered 12 statistical features from raw signals, then trained Logistic Regression, Random Forest, and XGBoost for both binary and multiclass classification. Random Forest achieved 98.1% accuracy with 95% seizure recall, missing only 23 out of 460 seizures — critical for patient safety in medical monitoring systems.
